## Features

- Using free GPT API from Groq
  | 使用 Groq 的免費 GPT API

- With a basic admin portal to view and edit FAQ
  | 管理後台，查看和編輯 FAQ

- The prompt is based on FAQ stored in database
  | GPT Prompt 是根據資料庫中的 FAQ 進行生成
